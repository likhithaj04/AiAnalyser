# LinkedIn + Resume Analyzer

A full-stack MERN application that analyzes resumes and LinkedIn profiles using AI-powered insights through Groq API. Users can upload resumes in PDF format or submit LinkedIn profile details with profile images to receive ATS scores, section-wise feedback, and personalized recommendations for improving professional profiles.

##  Features

### Resume Analysis
- Upload resumes in PDF format
- Extract content automatically from uploaded PDFs
- AI-powered resume analysis using Groq API
- Generate ATS (Applicant Tracking System) score
- Section-wise evaluation and feedback
- Identify strengths and missing areas
- Personalized suggestions for improvement

### LinkedIn Profile Analysis
- Analyze LinkedIn profile information
- Upload profile photo
- Evaluate profile quality and completeness
- AI-generated recommendations
- Improve professional visibility and presentation

### AI Insights
- Skill recommendations
- Profile enhancement suggestions
- Resume optimization feedback
- Content quality analysis
- Career-focused insights

### User Experience
- Responsive design
- Easy file upload interface
- Dynamic result generation
- Clean and intuitive UI
